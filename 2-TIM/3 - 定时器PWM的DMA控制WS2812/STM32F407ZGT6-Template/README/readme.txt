添加功能--WS2812

WS2812的RGB顺序为GRB

WS2812的高电平为 "1" == 高电平580ns-1000ns + 低电平580ns-1000ns		周期范围为:1160ns - 2000ns
WS2812的低电平为 "0" == 高电平220ns-380ns + 低电平580ns-1000ns		周期范围为:800ns - 1380ns
																	取中间值周期设为 1250ns 好计算其他值在范围内都可以															
WS2812复位信号24us

1250ns对应的频率为800kHz

开发板定时器的时钟为84Mhz 设置不分频 计一个数的时间为 1/84Mhz = 0.0000000119047s = 11.9047ns 

一个WS2812时序周期时间为 1/800khz = 0.00000125s = 1250ns;

1250ns/11.9047 = 105;

所以设定计数值为105

假定输出"1"  对应计数值为 52  高电平时间为 625ns 输出"1"的低电平时间为 625ns  
假定输出"0"  对应计数值为 28  高电平时间为 333ns 输出"1"的低电平时间为 917ns
